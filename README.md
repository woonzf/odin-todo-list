# Todo-list
### Live Demo: https://woonzf.github.io/odin-todo-list/
### Description:
- A web Todo application.
- Allows you to add / delete projects and tasks, mark tasks as "DONE".
- Created as a project of The Odin Project's Javascript Course

### Languages used:
- HTML
- CSS
- JavaScript

### Module Bundler:
